Connect Google Sheet
StrategyHub reads proposals from your Google Sheet. Setup takes ~5 minutes.
1
Create your Google Sheet
Go to sheets.new and add these headers in Row 1:

Title Category Type Audience Client Date Author Summary Activities Insights Outcome Thumbnail YouTube
Thumbnail — paste a public image URL (e.g. from Google Drive: share → copy link → convert to direct link)
YouTube — paste the full YouTube URL (e.g. https://youtu.be/abc123 or https://www.youtube.com/watch?v=abc123)
2
Share as "Anyone with the link — Viewer"
Click Share → Change to anyone with the link → Viewer → Done
3
Get your Sheet ID from the URL
URL: spreadsheets/d/SHEET_ID/edit
4
Get a Google API Key
console.cloud.google.com → Enable Google Sheets API → Credentials → Create API Key
