Connect your Google Sheet
StrategyHub reads your proposals directly from Google Sheets. Follow the steps below — takes about 5 minutes.
1
Create your Google Sheet
Open Google Sheets and create a new spreadsheet. Name it StrategyHub.

Add these exact column headers in Row 1:

Title  Category  Type  Audience  Client  Date  Author  Summary  Activities  Insights  Outcome
2
Make the sheet publicly readable
Click Share (top right) → Change to anyone with the link → set to Viewer. This lets the app read your data without login.
3
Get your Sheet ID
Your Sheet URL looks like:
https://docs.google.com/spreadsheets/d/SHEET_ID_HERE/edit

Copy the long ID between /d/ and /edit and paste it below.
4
Get a free Google API Key
Go to Google Cloud Console → Create credentials → API key.
Then enable the Google Sheets API for your project.
Paste your API key below.
